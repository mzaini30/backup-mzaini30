title: Custom domain dari Github Pages
---

Di repositori Githubnya, buat file bernama `CNAME` yang berisikan:

```
situs.com
```

Lalu, di penyedia layanan domain, setting DNS yaitu tambah A records dengan nilai-nilai berikut:

```
185.199.108.153
185.199.109.153
185.199.110.153
185.199.111.153
```
